# DSA Learning Repository in C

This repository is a learning resource for Data Structures and Algorithms (DSA) implemented in C. It’s designed to be easy to understand and extend.

## Build and Run
1. Clone the repo: `git clone <repo-url>`
2. Build the tests: `make all`
3. Run the array tests: `./tests/arrays/array_test`
